

# Name
ROCK LOVE HOUSE

# Description
音楽イベントでの体験の共有をサポートするアプリです。
具体的には音楽イベントでのセットリスト（アーティストが演奏した曲順）を投稿することができ、それを見ることができます。
将来的には、InstagramやTwitterに投稿するのを助けるイベントごとの画像投稿機能や、行けなくなったイベントチケットの取引が定価でできる機能を付ける予定です。
